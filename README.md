# README



Rails App for outputting fullscreen images/video to raspberry pi for use of electronic menuboards/advertisements.

Environment:
Ubuntu 16.04
Nginx
Puma
PostgreSQL

Prequistes:

LibreOffice:
```sudo apt install libreoffice```

MuPDF:
```sudo apt-get install mupdf mupdf-tools```
